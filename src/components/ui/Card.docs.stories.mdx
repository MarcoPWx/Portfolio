## Visual options

- hover: enables slight scale and elevated shadow on hover
- gradient: adds a subtle background gradient
- blur: applies backdrop blur for glass-like surface
- padding: one of `none | sm | md | lg`

## Composition patterns

- Place headings and actions inside the content tree; Card is intentionally unopinionated.
- For clickable cards, consider wrapping Card in a link, or ensure interactive children are focusable.
- For lists and grids, keep consistent padding and spacing.

## Accessibility guidelines

- Card renders a div; give landmarks or headings inside for screen reader context.
- Avoid putting multiple interactive elements inside a single clickable card without clear focus states.
- Respect color contrast for text placed on gradients or blurred surfaces.
