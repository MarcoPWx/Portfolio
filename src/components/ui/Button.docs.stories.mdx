## Usage

<Source
  code={`import { Button } from '@/components/ui/Button';
import { ArrowRight } from 'lucide-react';

export function Example() {
  return (
    <div className="flex gap-3">
      <Button variant="primary">Primary</Button>
      <Button variant="secondary">Secondary</Button>
      <Button icon={ArrowRight} iconPosition="right">Continue</Button>
    </div>
  );
}`}
  language="tsx"
/>

## Variants and sizes
- Variants: primary, secondary, ghost, danger, success
- Sizes: sm, md, lg

<Source
  code={`<Button variant="primary" size="md">Save</Button>
<Button variant="ghost" size="sm">Cancel</Button>
<Button variant="danger" size="lg">Delete</Button>`}
  language="tsx"
/>

## Icons
- Pass a Lucide icon component via the `icon` prop.
- Choose placement with `iconPosition` (left or right).

<Source
  code={`<Button icon={ArrowRight} iconPosition="left">Next</Button>
<Button icon={Download} iconPosition="right">Download</Button>`}
  language="tsx"
/>

## Loading
- Set `isLoading` to true to show a spinner and prevent interactions.

<Source
  code={`<Button isLoading>Submitting...</Button>`}
  language="tsx"
/>

## Accessibility guidelines
- Prefer `type="button"` for non-submit actions within forms.
- Use the native `disabled` attribute for non-interactive state.
- Ensure button labels are descriptive and unique.
