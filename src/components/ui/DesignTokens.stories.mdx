## Gradients

- Primary gradient: from `#10B981` to `#14B8A6` (`bg-gradient-to-r from-green-600 to-teal-600`)
- Secondary gradient surfaces are used with low opacity overlays for subtle depth

## Typography

- Font family (headings and body): Inter
- Code font: JetBrains Mono

### Scale (example)

- Heading 1: 2.25rem – 3rem (responsive)
- Heading 2: 1.875rem – 2.25rem
- Heading 3: 1.5rem – 1.875rem
- Body: 1rem – 1.125rem
- Small: 0.875rem

## Spacing & Radius (Tailwind)

- Spacing: `0.5, 1, 1.5, 2, 3, 4, 6, 8` (rem-based via Tailwind)
- Radius: `rounded, rounded-lg, rounded-xl, rounded-2xl`
- Shadows are mostly applied ad-hoc for 3D effects; prefer consistent elevation or subtle borders
