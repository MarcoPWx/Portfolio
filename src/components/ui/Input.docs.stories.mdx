## Error messaging and accessibility

- When showing an error, set `aria-invalid` on the control and connect the message via `aria-describedby`.
- You can pass `id` to the input and an `id` to the error element to link them.

<Source
  code={`export function FormField() {
  const id = 'email';
  const errorId = id + '-error';
  const hasError = true;

  return (
    <div className="space-y-1">
      <label htmlFor={id} className="block text-sm font-medium text-gray-300">Email</label>
      <div className="relative">
        <Input
          id={id}
          type="email"
          placeholder="you@example.com"
          aria-invalid={hasError}
          aria-describedby={hasError ? errorId : undefined}
        />
        {hasError && (
          <div id={errorId} className="mt-1 flex items-center gap-1 text-sm text-red-400">
            <AlertCircle size={14} /> <span>Enter a valid email.</span>
          </div>
        )}
      </div>
    </div>
  );
}`} language="tsx" />

## Icons

Pass a Lucide icon component via the `icon` prop and choose placement with `iconPosition`.

</div>`}
