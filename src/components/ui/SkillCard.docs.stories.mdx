SkillCard is a composite surface for showcasing skills, proficiency, years of experience, and relevant projects.
Supporting primitives like SkillGrid, SkillProgress, SkillStat, and CategoryHeader help compose rich layouts.

## Usage

<Source
  code={`import { SkillCard, SkillGrid, SkillProgress, SkillStat, ExpandableSkillCard, CategoryHeader } from '@/components/ui/SkillCard';

export function Skills() {
  return (
    <div className="space-y-8">
      <CategoryHeader icon={<span>⚙️</span>} title="Frontend" subtitle="UI engineering" />

      <SkillGrid columns={2}>
        <SkillCard
          title="React"
          level="Expert"
          years={6}
          description="Hooks, concurrent features, performance, and testing."
          projects={['Dashboards', 'Animations']}
          additionalTags={['Perf', 'A11y']}
        />
        <SkillCard
          title="TypeScript"
          level="Advanced"
          years={5}
          description="Types across app and packages; DT authoring and generics."
          projects={['Libraries', 'APIs']}
        />
      </SkillGrid>

      <div className="grid md:grid-cols-2 gap-4">
        <SkillProgress percentage={85} />
        <SkillStat label="Components shipped" value={"240+"} />
      </div>

      <ExpandableSkillCard
        title="Next.js"
        level="Advanced"
        years={4}
        description="App router, SSR/SG, performance optimization, ISR."
        expandedContent={<div>SEO, Edge, Streaming, RSC patterns, a11y.</div>}
      />
    </div>
  );
}`} language="tsx" />
