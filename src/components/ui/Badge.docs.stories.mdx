## Variants and sizes

Variants: default, success, warning, error, info, purple

Sizes: sm, md, lg

## Accessibility guidelines

- Use concise, descriptive text. Prefer text over icon-only badges.
- Ensure sufficient contrast between text and background.
- Avoid relying on color alone to convey meaningâ€”combine color with text and/or icon.
